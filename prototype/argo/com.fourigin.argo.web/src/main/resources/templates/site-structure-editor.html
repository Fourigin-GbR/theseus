Unused content

This lists content fields, who belong to a language which is not used / configured in the current project.

You can anyhow edit and delete it here.

Show favorites

You can enable / disable to show the favorite content fields of the current page/template.

The favorites are defined by the template settings.

Page path query

Here you see the full page of the current page inside the whole site-structure.

It is not using the display-names (page-names), as you see it in a opage-navigation!

You can click here also on every folder and directly will jump to that folder, in there to the first page.

Change state

Every page has a state: It can be live, in editing or not staged.

Only persons with the matching permissions can change the state.

If you make changes on a page, they have to chnaged the status to live.
